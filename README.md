# High-Resolution Soil Total Phosphorus Mapping for the Conterminous United States

## Overview

This repository serves as the project homepage for the study:

**High-Resolution Soil Total Phosphorus Mapping for the Conterminous United States Using Machine Learning**

The project developed national-scale machine learning models to predict soil total phosphorus across the conterminous United States by integrating soil, climate, topographic, land-use, and geospatial datasets.

The resulting maps provide a harmonized view of soil phosphorus distribution to support agricultural sustainability, nutrient management, and environmental decision-making.

---

## Research Highlights

- National-scale soil phosphorus mapping across the conterminous United States
- Integration of multi-source environmental datasets
- Random Forest machine learning for spatial prediction
- Geospatial analysis and predictive mapping in R
- Applications for sustainable nutrient management and environmental monitoring
